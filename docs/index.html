FEATURE_REGISTRY

Everything you need to lurk smart.

Built for streamers, fans, and drop hunters. Zero configuration, maximum automation.

AUTO_OPEN

Opens streams instantly

Opens tabs automatically the moment a channel goes live. No manual checking, no missed streams.

FOCUS_SPOOF

Invisible to platforms

Keeps streams active even when minimized. Platforms always think you're actively watching.

SMART_MUTE

Silent but counted

Muted by default, but audio is logically enabled. Drops and watch-time always register correctly.

MULTI_PLATFORM

One app, all platforms

Track channels on Twitch, YouTube, and Kick from a single unified interface. No switching required.

AUTO_CLOSE

Graceful cleanup

Closes tabs when streams end, with a configurable grace period (default 90s) to handle brief disconnects.

BANDWIDTH_SAVE

Low resource mode

Low-bandwidth mode keeps resource usage minimal while maintaining reliable stream detection and tracking.
